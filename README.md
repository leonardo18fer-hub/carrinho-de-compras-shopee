🛒 Carrinho de Compras estilo Shopee (Node.js + ECMAScript Modules)

Projeto desenvolvido utilizando JavaScript moderno (ECMAScript) com Node.js e modularização (ES Modules), simulando a lógica de um carrinho de compras inspirado na Shopee.

📌 Sobre o Projeto

Este projeto foca na lógica de negócio do carrinho, utilizando:

✅ ECMAScript moderno (ES6+)

✅ Node.js

✅ Modularização com import / export

✅ Separação de responsabilidades

✅ Boas práticas de organização de código

O objetivo é praticar estruturação de projetos backend e lógica de e-commerce.
